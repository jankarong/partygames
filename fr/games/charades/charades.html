🎯 Comment Utiliser le Générateur de Charades

  1. Sélectionnez les paramètres du générateur de charades
  2. Générez les mots de notre générateur de mots
  3. Agissez le mot généré (pas de paroles !)
  4. Laissez les autres deviner le mot de charades
  5. Marquez comme correct ou passez pour générer de nouveaux mots

⚠️ Rappelez-vous

  • 🤫 Pas de paroles
  • ✍️ Pas d'épellation
  • 👆 Ne pas pointer les choses
  • 🎭 Actions uniquement
