Comment jouer

Le principe est simple : lisez une phrase, puis répondez honnêtement.

1. Cliquez sur "Démarrer", puis sur "Suivant" à chaque tour.

2. Si vous l'avez déjà fait, prenez une gorgée ou baissez un doigt.

3. Si vous ne l'avez jamais fait, ne faites rien.

4. Continuez plusieurs tours pour faire monter l'ambiance.

5. Respectez les limites de chacun pour garder une partie fun.

Pourquoi cette page plaît

Cette version française de Jamais je n'ai jamais est rapide à lancer, mobile friendly et parfaite pour les soirées entre amis, les couples et les groupes adultes.
