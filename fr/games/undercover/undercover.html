Votre Rôle

Le Mot Secret

Règles

  • Un joueur est l'Agent Secret, les autres connaissent le mot
  • Les joueurs donnent des indices sur le mot
  • L'Agent Secret essaie de deviner le mot
  • Après chaque tour, les joueurs votent pour éliminer quelqu'un
  • Si l'Agent Secret est éliminé, les autres gagnent
