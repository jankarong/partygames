Règles du Jeu

Aperçu

Mafia est un jeu de déduction sociale palpitant où les joueurs sont secrètement divisés en deux équipes : la Mafia et les Villageois. Ce jeu excitant alterne entre les phases de nuit et de jour jusqu'à ce qu'une équipe gagne.

Rôles Disponibles

  • Mafia : Joueurs malveillants qui connaissent l'identité des autres (1-3 joueurs)
  • Villageois : Citoyens ordinaires essayant d'identifier la Mafia
  • Docteur : Villageois spécial qui peut sauver un joueur chaque nuit (0-1 joueur)
  • Détective : Villageois spécial qui peut enquêter sur un joueur chaque nuit (0-1 joueur)

Déroulement du Jeu

  1. Phase de Configuration : Chaque joueur reçoit une carte de rôle secrète
  2. Phase de Nuit : La Mafia élimine un joueur, le Docteur sauve quelqu'un, le Détective enquête
  3. Phase de Jour : Les joueurs discutent et votent pour éliminer un suspect

Conditions de Victoire

  • Les Villageois Gagnent : Quand tous les membres de la Mafia sont éliminés
  • La Mafia Gagne : Quand le nombre de Mafia égale ou dépasse le nombre de Villageois

Conseils pour Jouer

  • Observez attentivement le comportement des joueurs
  • Les rôles spéciaux doivent être prudents en révélant leur identité
  • Utilisez strategiquement vos votes
  • Ce jeu fonctionne mieux avec un groupe cohérent
