Statut des Joueurs

Règles du Jeu

Aperçu

Mafia est un jeu de déduction sociale palpitant où les joueurs sont secrètement divisés en deux équipes : la Mafia et les Villageois. Ce jeu excitant alterne entre les phases de nuit et de jour jusqu'à ce qu'une équipe gagne.

Rôles Disponibles

  • Mafia : Joueurs malveillants qui connaissent l'identité des autres (1-3 joueurs)
  • Villageois : Citoyens ordinaires essayant d'identifier la Mafia
  • Docteur : Villageois spécial qui peut sauver un joueur de l'élimination ET empoisonner un joueur pendant le jeu (0-1 joueur)
  • Détective : Villageois spécial qui peut enquêter sur l'identité d'un joueur chaque nuit (0-1 joueur)

Déroulement du Jeu

  1. Phase de Configuration :
    • Chaque joueur reçoit une carte de rôle secrète
    • Les joueurs doivent garder leurs rôles secrets sauf s'ils sont éliminés
  2. Phase de Nuit :
    • Tous les joueurs « dorment » (ferment les yeux)
    • Les membres de la Mafia se réveillent et choisissent un joueur à éliminer
    • Le Docteur se réveille et peut choisir de sauver ou d'empoisonner un joueur
    • Le Détective se réveille et peut enquêter sur l'identité d'un joueur
  3. Phase de Jour :
    • Tous les joueurs « se réveillent »
    • Les résultats des actions nocturnes sont révélés
    • Les joueurs discutent et débattent sur qui pourrait être la Mafia
    • Les joueurs votent pour éliminer un suspect supposé être de la Mafia

Capacités Spéciales

  • Pouvoirs du Docteur :
    • Peut sauver un joueur (y compris lui-même) une fois par jeu
    • Peut empoisonner un joueur une fois par jeu
    • Doit choisir entre sauver, empoisonner ou passer chaque nuit
  • Pouvoir du Détective :
    • Peut enquêter sur un joueur chaque nuit
    • Apprend si le joueur enquêté est de la Mafia ou non

Conditions de Victoire

  • Les Villageois Gagnent : Quand tous les membres de la Mafia sont éliminés
  • La Mafia Gagne : Quand le nombre de Mafia égale ou dépasse le nombre de Villageois

Conseils pour Jouer à Mafia

  • Observez attentivement le comportement des joueurs pendant les discussions dans ces jeux stratégiques
  • Les rôles spéciaux (Docteur et Détective) doivent être prudents en révélant leur identité
  • Utilisez strategiquement vos votes - parfois ne pas voter peut être la meilleure stratégie
  • Les membres de la Mafia doivent essayer de se mélanger avec les villageois pendant les discussions
  • Ce jeu fonctionne mieux avec un groupe cohérent pour développer des stratégies
